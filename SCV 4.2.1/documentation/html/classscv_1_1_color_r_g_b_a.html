void scv::ColorRGBA::toHLS ( float  hls[])

Converts the RGB color to HLS (Hue-Lightness-Saturation) format.

Parameters
hlsThe output color vector. Indices 0 through 2 will have the hue, saturation and lightness, respectively.

Example:

ColorRGBA color;
float hls_col[3];
color.toHLS(hls_col);
std::cout <<
"Hue: " << hls_col[0] <<
" Lightness: " << hls_col[1] <<
" Saturation: " << hls_col[2] << std::endl;
